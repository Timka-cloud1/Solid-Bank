# Bank app
## Instruction

    1) Download Solid_bank.jar
    
    2) Open terminal. 

    3) Enter "java -jar Solid_bank.jar"
    
    4) ENJOY AND HAVE FUN!!
    
## Operations
   
    "1 - show accounts"
   
    "2 - create account"
   
    "3 - deposit"
   
    "4 - withdraw"
   
    "5 - list of transactions"
   
    "6 - this message"
   
    "7 - exit"
